**🧠 What You Will Learn**

By using this cheat sheet, you will gain a strong understanding of the following C programming concepts:

---

**🚀 Quick Start & Compilation**

Learn how to write and execute your first C program:
- Structure of a basic C program  
- `#include <stdio.h>` and the `main()` function  
- Using `printf()` for output  
- Compiling programs using `gcc`  
- Enabling warnings with `-Wall` for better code quality  

This section helps you get started quickly with C development.

---

**📦 Data Types & Variables**

Understand how data is stored and managed in C:
- Primitive data types (`int`, `char`, `float`, `double`)  
- Memory size and ranges of data types  
- Declaring and initializing variables  
- Using `sizeof` operator  

This forms the backbone of all C programs.

---

**⌨️ Input / Output Operations**

Learn how to interact with users and the console:
- Output using `printf()`  
- Input using `scanf()`  
- Handling characters and strings  
- Safe input practices to avoid buffer overflow  
- Understanding why `gets()` is unsafe and better alternatives  

---

**🔀 Control Flow Statements**

Control the execution flow of programs using:
- `if`, `if-else`, and nested conditions  
- `switch-case` statements  
- Looping constructs:
  - `for` loop  
  - `while` loop  
  - `do-while` loop  

These concepts help in decision-making and repetition in programs.

---

**📚 Arrays**

Learn how to work with collections of data:
- Declaring and initializing arrays  
- Traversing arrays using loops  
- Calculating array size using `sizeof`  
- Working with one-dimensional arrays  

Arrays are essential for handling multiple values efficiently.

---

**🧵 Strings & String Functions**

Understand how strings work in C:
- Character arrays and string initialization  
- Common string operations using `<string.h>`  
- Important string functions:
  - `strlen`, `strcpy`, `strncpy`  
  - `strcat`, `strncat`  
  - `strcmp`, `strncmp`  
  - `strchr`, `strrchr`, `strstr`, `strtok`  

This section is crucial for text processing.

---

**👉 Pointers**

Learn one of the most powerful concepts in C:
- Pointer declaration and initialization  
- Address-of (`&`) and dereference (`*`) operators  
- Modifying values using pointers  
- Understanding memory-level access  

Pointers are fundamental for memory management and advanced programming.

---

**🧮 Functions**

Learn how to modularize your code:
- Function declaration and definition  
- Function calls and return values  
- Passing arguments to functions  
- Call by value vs call by reference  
- Swapping values using pointers  

Functions help in writing clean and reusable code.

---

**🏗️ Structures & typedef**

Group related data using:
- `struct` for creating custom data types  
- Accessing structure members  
- Initializing structures  
- Using `typedef` to create aliases for data types  

Structures are widely used in real-world C applications.

---

**📁 File Input / Output**

Learn how to work with files:
- Opening files using `fopen()`  
- Writing to files using `fprintf()`  
- Reading files using `fgets()`  
- Closing files properly with `fclose()`  

This section introduces persistent data storage.

---

**⚙️ Preprocessor Directives**

Understand compile-time instructions:
- `#define` for constants and macros  
- Conditional compilation using `#ifdef`  
- Header file inclusion  

Preprocessor directives improve flexibility and maintainability.

---

**🖥️ Command-Line Arguments**

Learn how programs receive input from the command line:
- `argc` and `argv` parameters  
- Iterating through command-line arguments  
- Practical use cases  

Useful for building configurable programs.

---

**🧠 Memory Management**

Understand dynamic memory handling:
- Allocating memory using `malloc` and `calloc`  
- Resizing memory using `realloc`  
- Releasing memory using `free`  
- Avoiding memory leaks  

This is a critical concept for low-level and systems programming.

**📊 Quick Reference Tables**

Handy reference tables for daily use:

- Escape sequences (`\n`, `\t`, `\\`, `\"`, `\'`)  

Ideal for fast lookup during coding.
